# Changelog

## [1.2.0] - 2026-01-24

### New Features
*   **Host Controls**: Improved host management capabilities within sessions. 
*   **Reaction System**: Introduced a new high-performance reaction animation system using particles.
*   **Mobile-First UI**: Complete refactor of the user interface for better mobile experience.

### Improvements
*   **WebRTC Reliability**: Enhanced connection stability with TURN server support and ICE restart logic.
*   **Performance**: Optimized animation system for better performance on mobile devices.
*   **Local Sync**: Improved Gmail subscription sync accuracy.

## [1.1.0] - 2026-01-20

### New Features
*   **Voice Rooms**: Added real-time audio calling for SharedDrop sessions.
*   **Mesh WebRTC**: Peer-to-peer audio topology supporting up to 6 participants.
*   **SQLite Persistence**: Added local database support for better data management.

### Improvements
*   **Automated Build System**: Streamlined build process for easier releases.
*   **Technical**: ICE configuration endpoint and Socket.IO signaling.

## [1.0.0] - 2026-01-10

*   Initial release of SharedDrop application.

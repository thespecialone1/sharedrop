## Getting Started

1. Download and install SharedDrop
2. Install [cloudflared](https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/downloads/)
3. Launch SharedDrop and select a folder to share
4. Share the link and password with your guests
5. Stop sharing anytime with one click
